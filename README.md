## How to add new Post
	- go to /admin.html
	- add post title
	- add post data
	- press compile button
	- copy under text
	- paste copied text in to db to - last elem DB.posts

### Data types

"# " - header text

"!sample.png" - image

"!https://www.youtube.com/embed/2Kbu?si=RC8L" - youtube insert video link

"- " - number list

"* " - point list

"\[Link Name\]\(Link\)" - a tag
